JavaScript Basics

    1. Introduction to JavaScript: JavaScript is a versatile, powerful, fast programming language. It is natively implemented in all modern web browsers, eliminating the need for additional plugins. The primary use of JavaScript is to add interactivity and dynamically update content on web pages.
    2. Variables: Variables in JavaScript act as containers to hold data values. Declaration can be made using the following keywords: 'var', 'let', or 'const'. For example, let myVariable = 'Bob'; declares a variable named 'myVariable' and assigns it the string value 'Bob'.
    3. Data Types: JavaScript supports various data types, including Number, String, Boolean, Object, Null, and Undefined. These data types allow for storing and manipulating different kinds of information within a JavaScript program.
    4. Operators: Operators in JavaScript perform operations on values. These include arithmetic operators (+, -, *, /, %) for calculations, assignment operators (=) to assign values to variables, and comparison operators (==, ===, !=, !==, <, >, <=, >=) to compare values.
    5. Functions: Functions in JavaScript are blocks of reusable code designed to execute specific tasks. A function is defined using the 'function' keyword, followed by a unique name, and parentheses (). Code awaiting execution by the function is enclosed within curly brackets {}.
    6. Control Structures: Control structures like conditional 'if...else' statements and loops ('for', 'while', 'do...while') are used in JavaScript to control the program's flow based on certain conditions.
    7. Objects: JavaScript is an object-oriented language. Almost everything in JavaScript is an object, including strings, arrays, dates, math instances, regular expressions, and objects themselves. This allows for structured and flexible data manipulation.
    8. Arrays:: An array in JavaScript is a special type of variable capable of holding multiple values simultaneously. An array is created using square brackets [], with individual items separated by commas.
    9. Events: Events are key in making JavaScript interactive. They represent actions or occurrences that can be detected and responded to by JavaScript code. Examples include user interactions such as clicks or key presses, or system-generated events like page loading.
    10. Console Logging: The console.log() function in JavaScript is a powerful debugging tool. It outputs any number of arguments to the console, which can be objects, arrays, numbers, strings, etc. Logging an object actually logs a reference to that object, not a snapshot. This means that if the object changes after it's logged, the updated values are shown in the console.

JavaScript Data Structures

    1. Dynamic Typing: JavaScript is a language with dynamic typing, allowing you to declare variables without specifying their data type. Also, data types can change within the lifetime of a variable.
    2. Primitive Values: There are seven data types in JavaScript that are considered primitive, which means they are not objects and have no methods. They are undefined, null, boolean, string, symbol, number, and bigint. All primitives are immutable, i.e., they cannot be altered.
        Undefined: When a variable hasn't been assigned a value, it is considered undefined.
        Null: This special keyword denotes a null or "empty" value.
        Boolean: A boolean represents a logical entity and can hold either the value true or false.
        String: Represents a sequence of characters.
        Symbol: A unique and immutable data type that can be used as an identifier for object properties.
        Number: Represents numeric values.
        BigInt: This data type was introduced in newer versions of JavaScript (ES2020) to represent integers of arbitrary length.
    1. Objects: In JavaScript, objects can be thought of as collections of properties. Each property establishes a connection between a name (or key) and a value. If the value is a function, the property is referred to as a method.
    2. Date: JavaScript Date objects represent a single moment in time in a platform-independent format. Date objects contain a Number that represents milliseconds since 1 January 1970 UTC.
    3. Array: An array in JavaScript is a high-level, list-like object that is used to represent ordered sets of data. It's a type of object for storing sequences of values.
    4. Typed Arrays and ArrayBuffers: JavaScript-typed arrays are objects similar to arrays that enable manipulation of raw binary data stored in memory. ArrayBuffer objects are used to represent a generic, fixed-length raw binary data buffer.
    5. Map and Set: JavaScript's Map and Set objects are collections that allow the storage and retrieval of data. A Map is a collection of keyed data items, just like an Object, but offers more memory-efficient and fast key lookups. A Set is a collection of unique values.
    6. WeakMap and WeakSet: These data structures are similar to Map and Set, but they do not prevent their elements from being garbage-collected. This feature becomes valuable in scenarios where you do not have control over the lifetime of the object.
