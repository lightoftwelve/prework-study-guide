CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style:

      Internal CSS style sheet
      External CSS style sheet

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Don't delete .DS_Store as its for indexing. This error arises:
      On branch feature/starter-code
      Changes not staged for commit:
        (use "git add ..." to update what will be committed)
        (use "git restore ..." to discard changes in working directory)
        modified: .DS_Store
      no changes added to commit (use "git add" and/or "git commit -a")

    FIX FOR THIS ERROR
    There are three options to fix it:
      Commit change: git commit -m "message"
      Stash change: git stash or do merge & then stash: git stash pop
      Discard local changes: git reset --hard or git checkout -t -f remote/branch

    Source: How to Resolve "Commit your changes or stash before merge":
